New in version 0.6

The following items were changed in this version:

  2. Started the UI look for 'demo' mode.

New in version 0.5

Significant overhaul to the entire system. Some of the changes include:

  1. Much faster processing of events.
  2. Only broadcast keys when one of the 'watched' apps is focused.
  3. Implentation of the black-list keys.
  4. Removed parts of UI not yet implemented.

New in version 0.4.1

The following items were changed in this version:

  1. Last window position is now remembered.
  2. Fixed the 'crash' issue where the 'a' key is registered by default.

New in version 0.4

The following items were changed in this version:

  1. Added native support for 64-bit.
  2. Polished UI code for the 'general settings'.
  3. Major infrastructure re-write changing the way events are handled.
